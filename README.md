# 💰 Pocket Track – Personal Finance App

Pocket Track is a modern personal finance tracking app built with Flutter.
It helps users manage their income, expenses, and financial insights in a simple and intuitive way.

---

## 🚀 Features

* 📊 **Dashboard Overview**

    * Current balance
    * Total income & expenses

* 💸 **Transaction Management**

    * Add, edit, delete transactions
    * Categorize income and expenses
    * Real-time updates

* 📈 **Insights**
    * Category-wise breakdown
    * Weekly spending trends
    * Weekly comparison chart

* 👤 **User Profile**

    * Add and edit personal details
    * Profile image support
    

* ⚙️ **Settings**

    * Theme selection (Light / Dark / System)
    * Logout with full data reset

---
